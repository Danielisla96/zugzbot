# 📜 Changelog - Zugzbot SDD

## [1.4.0] - 2026-05-23

### ✨ Añadido
- **Cost Monitor Plugin**: Implementación inicial del monitor de costos TUI para OpenCode.
- **Visualización Reactiva**: Uso de SolidJS para actualizaciones en tiempo real del consumo de tokens y costos asociados.
- **Detección de Agentes**: Lógica de agregación para agrupar costos basándose en el nombre del agente en la sesión.

### 🔧 Mejoras
- Refactorización de la estructura de plugins para soportar múltiples módulos simultáneos.
- Actualización de tipos en el SDK de plugins para mejorar el autocompletado.
