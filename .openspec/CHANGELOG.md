# Changelog - Zugzbot

Todas las modificaciones notables de este proyecto serán documentadas en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] - 2026-05-23

### Added
- **Monitoreo de Subagentes**: Visualización jerárquica de subagentes en el TUI con identación visual.
- **Rastreo de Costos**: Acumulación recursiva de costos y tokens para todas las sesiones hijas.
- **Mascota ASCII Animada**: Zugz ahora parpadea cada 3 segundos en el Sidebar.

### Fixed
- Recolección de métricas ahora soporta anidamiento profundo de sub-sesiones.
